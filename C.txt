Alumnos deconstruyendo Algoritmos (ADA) es una agencia privada de espionaje digital que está desarrollando un nuevo dispositivo para interceptar información usando ondas electromagnéticas, lo que permite espiar incluso sin contacto físico con el objetivo.

El dispositivo intenta recolectar información un byte a la vez, esto es, una secuencia de 8 bits donde cada uno de ellos, naturalmente, puede tener un valor de 0 o 1. En ciertas situaciones, debido a la interferencia de otros dispositivos, la lectura no se puede realizar con éxito. En este caso, el dispositivo devuelve el valor 9 para el bit correspondiente, informando que la lectura no se pudo realizar.

Para automatizar el reconocimiento de la información que lee el dispositivo, se solicitó un programa que, basándose en la información leída, informe si todos los bits se leyeron con éxito o no. Tu tarea es escribir este programa.

Input
La entrada consiste en una sola línea, que contiene 8 enteros N1,N2,N3,N4,N5,N6,N7
 y N8
, que indican los valores leídos por el dispositivo (Ni
 es 0, 1 o 9 para 1≤i≤8
).

Output
Imprime una sola línea que contenga la letra mayúscula 'S' si todos los bits se leen con éxito; de lo contrario, imprime una sola línea que contenga la letra mayúscula 'F', que corresponde a un fallo.

Examples
InputCopy
0 0 1 1 0 1 0 1
OutputCopy
S
InputCopy
0 0 1 9 0 1 0 1
OutputCopy
F
